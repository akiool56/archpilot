# Complexity Rubric

Each adds complexity: login, payment, role permissions, admin dashboard, history, file upload, batch processing, model switching, data dashboard, third-party integration, mobile app, multi-tenant, team collaboration, public API, large file processing, workflow approval, PDF/DOCX export, multi-format input, database persistence, global language switch.

Low: 0-2. Medium: 3-4. High: 5+.

After feature selection, output complexity level, reason, v1 features, v2+ features, not-now features, reserved interfaces, and implementation order.

Do not delete future features. Move them to later iterations and reserve interfaces.
