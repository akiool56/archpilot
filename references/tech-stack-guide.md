# Tech Stack Guide

## Next.js + TypeScript + Tailwind + API Routes

Best for formal web app, portfolio, Vercel, one repo, future Supabase, Chinese UI/i18n, server-side LLM config. Default for resume/document/contract tools.

## Python + FastAPI + simple frontend

Best for Python users, file parsing, table processing, backend-first internal tools.

## Streamlit

Best for fastest demo, local personal tool, Python beginner. Not ideal for polished web product.

## Vue + FastAPI

Best for Vue users, admin/workflow tools, clearer frontend/backend separation.

Always give one primary recommendation and one fallback.
