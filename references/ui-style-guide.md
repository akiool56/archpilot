# UI Style Guide

UI style options: 简洁专业, 作品集风格, 企业 SaaS, 轻量清爽, 自定义。

Color options: 蓝色系, 绿色系, 紫色系, 黑白灰极简, 自定义。

Every UI design must define: page header, layout, input card, result panel, primary button, secondary actions, empty/loading/error states, and responsive behavior.

Do not generate unstyled HTML, default English copy for Chinese UI, monochrome bare demo unless chosen, or fake result content.
