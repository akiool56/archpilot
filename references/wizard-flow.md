# Wizard Flow

Ask one question at a time.

## Steps

1. Project purpose: A 自己练手 / B 作品集 / C 交付 / D 商业化 MVP / E 内部工具
2. Target user: A 自己 / B 学生求职者 / C 公司员工 / D 管理员审核 / E 外部客户
3. UI language: A 中文 / B 英文 / C 中英文切换
4. UI style: A 简洁专业 / B 作品集 / C 企业 SaaS / D 轻量清爽 / E 自定义
5. Color: A 蓝 / B 绿 / C 紫 / D 黑白灰 / E 自定义
6. Input data: A 文本 / B PDF Word / C Excel CSV / D 图片 / E 多种
7. Output result: A 报告 / B 建议 / C 分类标签 / D 表格结构化 / E Markdown下载
8. V1 features: A 推荐核心闭环 / B +文件上传 / C +历史 / D +上传+历史 / E 自定义
9. Later extension: A 登录历史 / B 导出 / C 批量 / D 后台统计 / E 支付额度
10. Core data object: A 文本文档 / B 简历JD / C 合同条款风险 / D 表格字段结果 / E 其他
11. Data persistence: A 不保存 / B 本地保存 / C 数据库 / D 只保存结果 / E 不确定
12. Sensitive data: A 无 / B 简历个人信息 / C 合同客户 / D 财务医疗学生 / E 不确定
13. Deployment: A 本地 / B Vercel / C 服务器 / D 内网 / E 不确定
14. AI coding tool: A Claude Code / B Cursor / C Codex / D Windsurf Trae / E 未定
15. Technical level: A 完全新手 / B 一点前端 / C Python / D 一点后端 / E 完整项目经验
16. Stack: A 推荐 / B Next.js / C FastAPI / D Streamlit / E Vue+FastAPI
17. LLM config: A 服务端 env / B 本地页面输入 / C 还没有 Key / D 指定供应商 / E 推荐
18. Comment preference: A 学习型详细 / B 维护型适中 / C 极简
19. Personalization: A 更好看 / B 方便学习 / C 稳定少 bug / D 易扩展 / E 自定义一句
20. Confirmation draft
