# Intake Template

Use only if the user asks for a full form. Default to wizard mode.

Fields: project idea, purpose, target user, UI language, UI style, color, input, output, v1 features, later extension, core data object, data persistence, sensitive data, deployment, AI coding tool, technical level, stack, LLM config, comment preference, personalization.

Defaults: Chinese UI for Chinese-facing apps, modular monolith, no login/payment/admin in v1, no mock-result mode, server-side real LLM path, no data saving unless needed, run checks after every stage.
