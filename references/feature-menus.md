# Feature Menus

## Resume rewriting

V1: paste resume, paste JD, real LLM analysis, missing keywords, issues, rewrite suggestions, copy Markdown, loading/error.

Later: PDF/Word upload, history, version management, PDF/DOCX export, paid quota, batch JD analysis, mock interview.

Not v1: auto apply, enterprise recruiter dashboard, payment, multi-tenant permissions.

## Document summarizer

V1: paste/upload one document, summary goal, real LLM summary, key points, action items, copy Markdown.

Later: cloud drive sync, multi-document knowledge base, team sharing, public API.

## Contract analyzer

V1: paste/upload contract, select type, real LLM key clause extraction, risk points, revision suggestions, disclaimer.

Later: template library, legal review workflow, DOCX memo export, team permissions.

## Spreadsheet cleaner

V1: upload Excel/CSV, preview fields, select cleaning goal, identify problems, generate suggestions, export CSV.

Later: batch files, saved rules, data quality score, scheduled jobs.
