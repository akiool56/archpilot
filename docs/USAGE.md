# Usage

Recommended call:

```text
请使用 archpilot skill。我想做一个 AI 简历改写工具。请按一步一步选择的方式，引导我确定需求、MVP、中文界面、UI风格、复杂度、模块职责、接口预留、真实 LLM 配置和分阶段编码方案。
```

Expected behavior: ask one choice question at a time, translate requirements, analyze complexity, produce confirmation draft, then produce 《AI 编程项目 LLM 开发设计包》 with detailed module contracts, change impact, staged coding prompts, and run checks.


## Direct invocation

```text
/archpilot 我想做一个 AI 简历改写工具。
```

或者：

```text
请使用 ArchPilot（架构领航）skill。我想做一个 AI 简历改写工具。
```
