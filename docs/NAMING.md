# Naming

## Public name

ArchPilot

## 中文名

架构领航

## Short tagline

把模糊想法变成 AI 可执行架构。

## Full positioning

ArchPilot is an LLM-executable architecture designer for AI coding beginners. It converts fuzzy app ideas into responsibility-complete, not technically overcomplicated, high-cohesion, low-coupling architecture packages with staged coding prompts.

## Recommended invocation

```text
/archpilot 我想做一个 AI 简历改写工具。
```

or:

```text
请使用 ArchPilot（架构领航）skill。我想做一个 AI 简历改写工具。
```
